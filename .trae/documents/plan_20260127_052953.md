# 全真模考页面重构方案

## 📋 核心变更

### 1️⃣ 数据源重构
- **查询 mock_exams 表**，获取预制的仿真试卷
- **读取字段**：
  - `rewritten_content`：AI 深度改写的学术文章
  - `questions`：真题级题目（双模博弈验证）
  - `analysis`：考后复盘解析

### 2️⃣ 沉浸式仿真考场 UI
- **全屏模式**：隐藏顶部导航栏和底部栏
- **左右分栏布局**：
  - 左侧（60%）：ScrollArea 显示文章内容
    - 字体：Times New Roman / Noto Serif
    - 字号：text-lg
    - 行高：leading-loose
  - 右侧（40%）：题目区域
    - 固定定位，独立滚动

### 3️⃣ 考试功能
- **倒计时器**：右上角悬浮，默认 15:00
- **高亮标记**：允许选中左侧文章文字
- **防作弊**：禁用右键菜单和复制功能

### 4️⃣ 考后反馈
- **交卷后切换**：立即进入"解析模式"
- **错题高亮**：右侧题目区显示 ✅/❌
- **深度解析**：每道错题下方展开 analysis 内容
- **能力图更新**：显示"逻辑推理能力 +5"

## 🛠️ 实现步骤

1. 创建页面文件 `app/practice/mock/page.tsx`
2. 实现 Supabase 数据查询逻辑
3. 构建沉浸式考场布局（左右分栏）
4. 实现倒计时器组件
5. 实现题目渲染和交互
6. 实现文章高亮功能
7. 实现防作弊 UI
8. 实现考后反馈系统
9. 添加能力图更新逻辑
