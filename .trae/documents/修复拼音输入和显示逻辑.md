## 修复拼音输入和显示逻辑

### 当前问题

1. **输入框提示显示中文** - 但用户应该输入拼音
2. **Pinyin 区域显示逻辑** - 需要显示正确答案的拼音

### 正确逻辑（LEARN_CHINESE 模式）

**显示区域：**
- 显示中文句子："你能帮我一下吗？"

**输入框：**
- 用户输入**拼音**（如 "ni", "neng", "bang", "wo", "yi", "xia", "ma"）
- 输入框数量 = 中文字符数
- 提示显示**拼音**而不是中文

**Pinyin 区域：**
- 显示**正确答案的完整拼音**（如 "nǐ néng bāng wǒ yí xià ma"）

### 需要修改的地方（应用到所有类别）

#### 1. 修改单词练习的输入框提示（第417-420行）

**当前：**
```tsx
const hintText = isLearnChinese 
  ? (currentItem as WordItem).zh[index] || ''  // 显示中文
  : (currentItem as WordItem).en[index] || ''
```

**改为：**
```tsx
// 学中文模式：输入框提示显示拼音；学英文模式：提示显示英文
const hintText = isLearnChinese 
  ? (currentItem as WordItem).pinyin.split(' ')[index] || ''  // 显示拼音
  : (currentItem as WordItem).en[index] || ''
```

#### 2. 修改句子练习的输入框提示（句子练习部分）

**当前：**
```tsx
value={showHint ? tokens[index] || '' : value}  // tokens 是中文
```

**改为：**
```tsx
// 学中文模式：提示显示拼音；学英文模式：提示显示英文
const hintTokens = isLearnChinese
  ? (currentItem as SentenceItem).pinyin.split(' ')
  : tokens
value={showHint ? hintTokens[index] || '' : value}
```

#### 3. 答案验证逻辑

需要更新答案验证，比较用户输入的拼音是否正确。

### 关键修改点（应用到所有类别）

1. **单词练习** - 输入框提示显示拼音片段
2. **句子练习** - 输入框提示显示拼音片段
3. **Pinyin 区域** - 始终显示完整拼音作为答案参考
4. **答案验证** - 比较用户输入的拼音与正确答案拼音
