# 将 /check-in 页面改为首页（遵循正确的语言逻辑）

## 语言逻辑确认

### LEARN_ENGLISH 模式（学英语）
- ✅ 全站 UI 显示英文（导航、按钮、提示）
- ✅ 学习页面：题目显示英文，辅助解析显示中文
- ✅ 阅读页面：内容显示中文

### LEARN_CHINESE 模式（学中文）
- ✅ 全站 UI 显示中文（导航、按钮、提示）
- ✅ 学习页面：题目显示中文，辅助解析显示英文
- ✅ 阅读页面：内容显示英文

## 修改内容

### 1. 修改 app/check-in/page.tsx
- 引入 `useLanguage` hook 和 `TRANSLATIONS`
- 根据语言模式显示对应语言的 UI 文本
- 添加首页内容：
  - 欢迎语（根据语言模式）
  - 签到功能（连绵、落墨、精进）
  - 今日学习进度
  - 快捷功能入口（修习、文库、我的）
  - 本周签到状态
- 所有文本使用 `t.nav.*`、`t.dashboard.*` 等翻译键

### 2. 修改 app/layout.tsx
- 更新页面 metadata title 为首页相关
- 确保浏览器标签页显示正确的首页名称

## 语言逻辑实现
- 使用 `const { learningMode } = useLanguage()`
- 使用 `const t = TRANSLATIONS[learningMode]`
- 所有 UI 文本使用 `t.xxx` 显示对应语言
- 确保首页遵循上述语言逻辑
