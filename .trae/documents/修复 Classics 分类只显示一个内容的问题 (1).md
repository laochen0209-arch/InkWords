# 修复 Classics 分类只显示一个内容的问题

## 问题分析

从数据库截图可以看到：

* 有很多 `category: 'classics'` 的文章

* 数据库中确实有数据

但页面只显示一个，可能的原因是：

1. **API 返回的数据问题** - 可能只返回了一条数据
2. **前端过滤逻辑问题** - 可能过滤条件不正确
3. **React key 问题** - 可能有重复的 key

## 修复计划

### 1. 检查 API 路由

确保 API 正确返回所有文章，而不是只返回一条

### 2. 检查前端过滤逻辑

确保过滤逻辑正确处理 classics 分类

### 3. 检查 React key

确保每个文章有唯一的 key

## 预期效果

* Classics 分类应该显示所有 category 为 'classics' 的文章

* 页面应该显示多个卡片，而不是只有一个
