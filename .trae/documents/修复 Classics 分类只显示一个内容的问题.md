# 修复 Classics 分类只显示一个内容的问题

## 问题分析

### 可能原因：
1. **数据库中的 category 字段值不匹配** - 可能使用了其他值如 'culture'、'history' 等
2. **API 返回的数据问题** - 可能没有正确返回所有文章
3. **前端过滤逻辑问题** - 过滤条件可能不正确

## 修复计划

### 1. 添加调试信息
在 API 路由和前端组件中添加 console.log 来查看实际数据

### 2. 检查过滤逻辑
确保过滤逻辑正确处理所有可能的 category 值

### 3. 验证数据流
确认从数据库 → API → 前端的数据流是否正确

## 预期效果
- Classics 分类应该显示所有 category 为 'classics'、'Classics'、'culture'、'history' 的文章
- 调试信息可以帮助定位问题
