# 修复登录数据库连接问题

## 问题原因
- `.env` 文件中有 `POSTGRES_URL`，但 Prisma 需要 `DATABASE_URL`
- 导致数据库连接失败，API 返回错误

## 修复步骤

### 1. 修改 .env 文件
添加 `DATABASE_URL` 环境变量，使用与 `POSTGRES_URL` 相同的值

### 2. 生成 Prisma Client
运行 `npx prisma generate` 生成 Prisma Client

### 3. 同步数据库结构
运行 `npx prisma db push` 将 schema 同步到数据库

### 4. 重启开发服务器
重启 `npm run dev` 使环境变量生效

### 5. 测试登录功能
验证登录是否正常工作
